## 技术变更详情

### 1. 数据库变更
- **升级前**：pgvector/pgvector:pg16
- **升级后**：paradedb/paradedb:latest-pg17
- **原因**：v2 需要 pg_search 插件，仅支持 PostgreSQL 17
- **数据迁移**：使用完整数据库备份恢复

### 2. 认证系统变更
- **升级前**：NextAuth.js
- **升级后**：Better Auth
- **SSO提供商**：继续使用 Casdoor（原生支持）
- **环境变量变更**：
  - 移除 NextAuth 特定变量
  - 添加 Better Auth 变量
  - 保留 Casdoor 变量：`AUTH_CASDOOR_ID`, `AUTH_CASDOOR_SECRET`, `AUTH_CASDOOR_ISSUER`

### 3. 应用版本变更
- **升级前**：v1.x.x
- **升级后**：v2.1.42
- **镜像**：docker.io/lobehub/lobehub:2.1.42

### 4. 配置变更
- **deploy-lobe.yaml**：更新环境变量和镜像版本
- **deploy-postgresql.yaml**：更新镜像和PVC引用
- **Secrets**：更新认证密钥（AUTH_SECRET, JWKS_KEY）
